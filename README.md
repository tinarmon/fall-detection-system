# การพัฒนาระบบตรวจจับการเสียการทรงตัวของบุคคลโดยใช้เทคโนโลยี MediaPipe
**Development of a Loss of Balance Detection System for Individuals using MediaPipe Technology**

โปรเจกต์นี้เป็นส่วนหนึ่งของโครงร่างปริญญานิพนธ์ มหาวิทยาลัยเทคโนโลยีราชมงคลล้านนา ตาก มีวัตถุประสงค์เพื่อพัฒนาระบบแจ้งเตือน "สภาวะก่อนการล้ม" (Pre-fall / Loss of Balance) ของผู้สูงอายุผ่านกล้องวิดีโอ โดยใช้การวิเคราะห์โครงร่างร่างกาย (Pose Landmark) แทนภาพบุคคลจริงเพื่อรักษาความเป็นส่วนตัว และประมวลผลด้วยโมเดลเรียนรู้เชิงลึกแบบอนุกรมเวลา (GRU) 

## 📂 โครงสร้างของโปรเจกต์ (Project Structure)

โปรเจกต์ถูกแบ่งออกเป็น 5 โมดูลหลักเพื่อให้ง่ายต่อการพัฒนาและดูแลรักษา:

- `pose_estimator.py`: โมดูลดึงภาพจากกล้องและสกัดพิกัดร่างกาย 33 จุดด้วย MediaPipe
- `angle_calculator.py`: โมดูลคัดกรองจุดเป้าหมาย 6 จุด (ไหล่, สะโพก, เข่า) และคำนวณความสัมพันธ์เชิงมุมของร่างกายซีกซ้ายและขวา
- `collect_data.py`: สคริปต์สำหรับบันทึกข้อมูลพฤติกรรมการเคลื่อนไหวและการเสียการทรงตัวลงในไฟล์ CSV
- `fall_dataset.csv`: ชุดข้อมูลอนุกรมเวลา (Time-series data) ที่ได้จากการเก็บข้อมูล
- `train_model.py`: สคริปต์สำหรับสร้างและฝึกสอนโมเดล Gated Recurrent Unit (GRU) ด้วย Keras/TensorFlow
- `fall_model.keras`: ไฟล์โมเดล AI ที่ผ่านการฝึกสอนและพร้อมใช้งาน
- `main.py`: โปรแกรมหลักสำหรับรันระบบตรวจจับและแจ้งเตือนการเสียการทรงตัวแบบเรียลไทม์
- `pose_landmarker_full.task`: โมเดลน้ำหนัก (Weights) ของ MediaPipe
- `requirements.txt`: รายการไลบรารีที่จำเป็นสำหรับโปรเจกต์นี้
